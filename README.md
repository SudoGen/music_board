## A Musical Instrument App
#### Made in Python

******

## Setup
- Installing all the requirements need to run the program.
```bash
python setup.py
```
or
```
python -m pip install requirements.txt
```
- For running the program
```
python main.py
```

> Click on blocks and explore the program | Enjoy The Music
